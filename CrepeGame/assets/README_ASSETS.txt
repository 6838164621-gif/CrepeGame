Put your real photos in this folder so the game can load them.

Supported file types:
- .png
- .jpg
- .jpeg
- .webp

Use these filenames for topping button images:
- chocolate
- banana
- whipped_cream
- strawberries
- nutella
- marshmallows
- oreos
- sprinkles
- condensed_milk
- blueberry_jam
- peanut_butter
- brownies
- foi_thong
- raspberries
- jelly

Optional extra background files:
- menu_background.png / .jpg
- game_background.png / .jpg
- plate.png / .jpg

Example:
assets/banana.png
assets/whipped_cream.jpg
assets/game_background.png
